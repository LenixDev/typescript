# FiveM TypeScript Template

## Setup

Install dependencies:
```bash
npm install
```

## Build

Build all projects (client, server, web):
```bash
npm run build
```

## Watch Mode

Auto-rebuild on file changes:
```bash
npm run watch
```

## Individual Builds

```bash
npm run build:client
npm run build:server
npm run build:web
```

## Individual Watchs

```bash
npm run watch:client
npm run watch:server
npm run watch:web
```
